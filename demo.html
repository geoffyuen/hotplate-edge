Another CSS 12-column grid system for modern browsers

Real gutters and real source ordering

We use flexbox, calc and margins to build the gird. Pad and use background colours without breaking your layouts. Use flexbox order rules instead of sloppy push and pull hackery.

No rows and no floats

No more extra markup to clear columns with clearfix classes or rules taking up your precious pseudo elements. Columns wrap automatically to the next line if they don't fit. This means cleaner markup.

Nesting!

Use columns for general layout like headers, content, sidebars and footers. User columns inside those sections to build sub-layouts.

Using flexbox means equal height columns

Finally you can implement your designer's layouts without any javascript for specifying heights with magic numbers.
